API Version discoverability level

By default, users who are not members of your organization can not find any of your APIs.

By changing the discoverability of this plan, you can selectively allow non-members to find and subscribe to your plans.

By applying this change, your API would be visible to:

  • Members with explicit view permissions in this organization.
  • Any user with an active Apiman account, including users in other organizations.
  • Developer portal users, including those browsing anonymously.
    • Further configuration for the developer portal can be found in the Developer Portal tab.
This API will be visible to users outside your organization.
