Role Management

Create and modify roles that users can be granted membership in for any organization. Each role grants the user a set of permissions, allowing her to do specific things within the organization.

New Role
Organization Owner Auto-granted to org creator
This role is automatically given to users when they create an organization. It grants all permissions.
Grants Permissions: Org View, Org Edit, Org Admin, Service View, Service Edit, Service Admin, Application View, Application Edit, Application Admin
Application Developer
This role allows users to perform standard application development tasks (manage applications but not services or plans).
Grants Permissions: Application View, Application Edit, Application Admin
Service Developer
This role allows users to perform standard service development tasks such as managing services and plans.
Grants Permissions: Service View, Service Edit, Service Admin
